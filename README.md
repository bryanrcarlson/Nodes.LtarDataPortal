"# Nodes.LtarDataPortal" 

### Purpose

Writes files formated to be ingested by NAL's LTAR data portal as outlined by [LTAR Common Observatory Meteorology Data Concept of Operations Version 2.0 | https://www.ars.usda.gov/ARSUserFiles/np211/LTAR%20Meteorology%20Concept%20of%20Operations%20FINAL%20150929.pdf]
